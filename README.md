# geometry_poster
Geometry cheat sheet made using latex tikzposter package and geogebra

## How to work with this project
geomposter.tex is the main file

test.tex is a playground for testing the geogebra exported figures

geogebra/ folder contains raw geogebra files (.ggb) as well as .txt files exported from geogebra as PGF/TikZ

Note: use geogebra classic version to export to tikz, (https://www.geogebra.org/classic) File -> Download As -> PGF/TikZ

figures/ folder contains raw tex files corresponding to every figure to include in the main document after some adjustments
